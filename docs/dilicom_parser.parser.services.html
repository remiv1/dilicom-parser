Submodules¶

dilicom_parser.parser.services.eancom module¶

Parseur de messages de service au format EANCOM D96A (APERAK).

dilicom_parser.parser.services.eancom.parse_eancom(content)¶

Parse le contenu brut d’un fichier EANCOM APERAK.

Le séparateur de segment est le retour à la ligne (après UNA). Les éléments sont séparés par “+”, les sous-éléments par “:”.

Type renvoyé:

EancomInterchange

dilicom_parser.parser.services.gencod module¶

Parseur de messages de service au format GENCOD (CNUT 05003).

dilicom_parser.parser.services.gencod.parse_gencod_lines(lines)¶

Parse les lignes d’un fichier GENCOD et retourne les messages de service.

Le fichier est structuré en blocs délimités par des CNUT (05003…199). Chaque bloc contient des CNUR identifiés par leur code rubrique.

Type renvoyé:

List[GencodServiceMessage]
